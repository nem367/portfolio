Generative Art: 60s aesthetic

This was created for my Creative Coding class in Spring 2022. In this project, I explore how to control generativity through limiting the randomness of colors picked, as well as the sizes of the shapes drawn. The result is an aesthetic art piece with colors that go well together and shapes that vary from one frame to another while still a having a similar aesthetic.

Vintage Digital Clock

This project was created for my Creative Coding class in Spring 2022. The task was to come up with an innovative way of telling the time.

The clock that I created draws different arcs for each hour in a day as minutes go by. Each arc represents an hour in the 24 hours of the day. The drawing starts from the inside to the outside and the bright pink outer arc represents the current hour being drawn every minute. The big dot is the current hour in the position it is in traditional analogue clocks. Click here for this project's documentation.

Sign your Zellige!

This was my Creative Coding final project (Spring 2022). For this project, I decided to use machine learning (Teachable Machine, tensorflow.js) to teach my computer to recognize colors signed in Moroccan sign language. As colors are signed into the camera, a p5.js code generates random zellige squares that follow the colors users sign. With each color that is signed, zellige squares appear in random shades of that color. Click here for this project's documentation and here for a presentation of this project.
